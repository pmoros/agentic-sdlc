# Scratchpad

This folder serves as a temporary workspace for drafts, experiments, and work-in-progress documentation.

## Purpose

Use this space for:
- Draft documents before formal publication
- Experimental ideas and prototypes
- Temporary notes and calculations
- Collaborative brainstorming content

## Guidelines

- Content here is considered temporary and should not be referenced in formal documentation
- Clean up regularly - move finalized content to appropriate folders
- Use descriptive filenames with dates when appropriate (e.g., `2026-02-11-draft-proposal.md`)
- This folder is not subject to formal review processes
